ทดสอบแอปพลิเคชัน 'เที่ยวชุมชน'

ดำเนินการทดสอบแอปพลิเคชัน 'เที่ยวชุมชน' ทั้งในรูปแบบ Manual และ Automated Testing เพื่อค้นหาข้อบกพร่อง (Bugs) และตรวจสอบการทำงานของฟีเจอร์ต่างๆ ให้เป็นไปตามที่ออกแบบไว้

เครื่องมือที่ใช้ (Tools Used):

  • Appium
  • visual studio code
  • Java (สำหรับเขียนสคริปต์ทดสอบ)
  • Googlesheet(สำหรับ Test Cases)
  • Python

ทดสอบแอปฯ PSRU Bus (Prompt Go)

ออกแบบและพัฒนาสคริปต์สำหรับทดสอบแอปพลิเคชัน PSRU Bus แบบอัตโนมัติ เพื่อตรวจสอบความถูกต้องของการแสดงตำแหน่งรถไฟฟ้าแบบเรียลไทม์ และทดสอบประสิทธิภาพของระบบ

เครื่องมือที่ใช้ (Tools Used):

  • Appium
  • visual studio code
  • Java (สำหรับเขียนสคริปต์ทดสอบ)
  • Googlesheet(สำหรับ Test Cases)
  • Python

กิจกรรมอบรมแอปฯ 'เที่ยวชุมชน'

ลงพื้นที่จัดกิจกรรม “อบรมถ่ายทอดความรู้การใช้งานแอปพลิเคชันท่องเที่ยวชุมชน” ให้แก่กลุ่มตัวอย่างผู้ใช้งานใน อ.คีรีมาส จ.สุโขทัย การอบรมเน้นการปฏิบัติจริง พร้อมแลกเปลี่ยนความคิดเห็นเพื่อนำมาปรับปรุงแอปพลิเคชัน

ทักษะที่ได้รับ:

  • การสื่อสารและการนำเสนอ
  • การเก็บรวบรวม Feedback จากผู้ใช้งาน
  • การทำงานร่วมกับชุมชน

ทัศนศึกษา AUTOMATION EXPO 2025

ทัศนศึกษาดูงานด้านเทคโนโลยีระบบอัตโนมัติและหุ่นยนต์ ณ ศูนย์ประชุมนานาชาตินงนุชพัทยา และ EEC Automation Park มหาวิทยาลัยบูรพา เพื่อเรียนรู้เทคโนโลยีอุตสาหกรรม 4.0

สิ่งที่ได้เรียนรู้:

  • เทคโนโลยี IIoT (Industrial Internet of Things)
  • ระบบแขนกลและหุ่นยนต์ในโรงงานอุตสาหกรรม
  • โซลูชันด้าน Automation สมัยใหม่
